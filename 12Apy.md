### Возрастает ли список?

Дан список. Определите, является ли он монотонно возрастающим (то есть верно ли, что каждый элемент этого 
списка больше предыдущего). Выведите YES, если массив монотонно возрастает и NO в противном случае.

__Решение с флагом__
```py
a = list(map(int, input().split()))
n = len(a)
flag = True
for i in range(1, n):
    if a[i - 1] >= a[i]:
        flag = False
        
if flag:
    print("YES")
else:
    print("NO")
```
