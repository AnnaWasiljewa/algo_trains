### Номер появления слова

Во входном файле (вы можете читать данные из файла _input.txt_) записан текст. 
Словом считается последовательность непробельных символов идущих подряд, 
слова разделены одним или большим числом пробелов или символами конца строки. 
Для каждого слова из этого текста подсчитайте, сколько раз оно встречалось в этом тексте ранее.

```cpp
#include <iostream>
#include <fstream>
#include <sstream>
#include <string>
#include <map>

int main() {
    std::ifstream inputFile("input.txt");
    if (!inputFile.is_open()) {
        std::cerr << "Ошибка: не удалось открыть файл input.txt" << std::endl;
        return 1;
    }

    std::map<std::string, int> wordCount;
    std::string line;

    while (std::getline(inputFile, line)) {
        std::istringstream iss(line);
        std::string word;

        while (iss >> word) {
            // Выводим текущее количество предыдущих вхождений слова
            std::cout << wordCount[word] << " ";

            // Увеличиваем счётчик для этого слова
            wordCount[word]++;
        }
    }

    inputFile.close();
    return 0;
}
```
Время: 106ms. Память: 7.41Mb.
